## About Webook
Facebook/Twitter clon [WORK IN PROGRESS]

## Stack
- Laravel 6.x
- Eloquent con lib para mongodb
- Boostrap-Vue
- Boostrap 4

## How to run it?
- npm install (dependencies)
- execute the mongodb server (mongod)(and if you really need change the settings in config/app.php - database.php)
- php artisan serve
